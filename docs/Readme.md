Project ini adalah website portofolio dinamis berbasis Laravel yang dibuat untuk memenuhi tugas UTS Pemrograman Web. Website ini menampilkan profil, skill, project, contact form, serta rencana perancangan sistem berupa ERD dan Flowchart.

Website ini menggunakan frontend berbasis Livewire dan Blade, admin panel menggunakan Filament V3, database MariaDB, serta berjalan menggunakan Docker.

---

## Teknologi yang Digunakan

- Laravel
- Livewire
- Blade
- Filament V3
- MariaDB
- Docker
- PHP
- CSS Custom tanpa NPM

---

## Fitur Utama

### 1. Halaman Frontend

Website memiliki halaman frontend yang menampilkan:

- Home / About
- Skill dan Tech Stack
- Featured Project
- All Projects
- Contact Form
- Diagram ERD dan Flowchart Sistem

Frontend dibuat menggunakan Livewire dan Blade.
